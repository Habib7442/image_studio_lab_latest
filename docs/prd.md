## 1. Executive Summary

**imagestudiolab.com** is a digital prompt-pack store solving one universal problem:

> *"I see beautiful AI images on social media. I want to make them. I can't write the prompts."*

**The product:** curated, copy-paste prompt packs that produce pro-quality outputs in **ChatGPT (GPT Image)** and **Gemini Nano Banana Pro** — no prompt engineering required.

Each pack ships with: a **Master Director system prompt**, **25 production-tested prompts** with Subtle/Medium/Dramatic intensity variants, **troubleshooting guide**, **iteration cheat sheet**, and **typography pairings** where relevant.

**The wedge:** unlike PromptBase (generic marketplace), Midjourney creators (require $30/mo subscription), or single-niche aesthetic packs, imagestudiolab serves **four distinct buyer segments under one universal brand**:

- 🛒 **Sellers** — Product Photoshoot, Lifestyle, Food, Real Estate
- 🎨 **Creators** — Memory Lane, Cinematic Lighting, Celebrity Photographer
- 🏏 **Fans** — Cricketer Selfie, IPL Team Packs, Bollywood Portrait
- 💼 **Professionals** — LinkedIn Headshot, Personal Brand, Album Cover

**Distribution:** Phase 1 leverages existing IG cricket audience (3K+ likes/reel on @designnai.official) as **trojan horse** for the cricket-segment packs. Pinterest, LinkedIn, and TikTok scale subsequent segments.

**Pricing:** $9–29 a la carte + **$99 Studio Vault Lifetime** (founder's pricing, capped at 250). Lifetime delivers via Gumroad's auto-update file mechanic — past buyers get every new pack automatically.

## 7. Pack Format & Quality Standard

Every pack ships in this exact structure (locked):

### Pack contents (38–40 pages PDF)

| Page | Content |
|---|---|
| 1 | Editorial cover |
| 2 | How to use (4-step ritual + tool comparison) |
| 3 | **Master Director system prompt** (paste once) |
| 4 | Quality boosters + iteration prompts |
| 5 | Section 1 cover |
| 6–10 | 5 prompts (with Subtle/Medium/Dramatic variants) |
| 11 | Section 2 cover |
| 12–16 | 5 prompts |
| 17 | Section 3 cover |
| 18–22 | 5 prompts |
| 23 | Section 4 cover |
| 24–28 | 5 prompts |
| 29 | Section 5 cover |
| 30–34 | 5 prompts |
| 35 | Troubleshooting (10 common AI failures + exact fixes) |
| 36 | Typography pairings (where applicable) |
| 37 | Pack specs / Back cover |

### Each prompt structure (locked)

- **Memorable title** (e.g., "The Glow Hour" not "Golden Hour Lighting")
- **Mood line** — one evocative italic sentence
- **Nano Banana Pro version** (~150–250 words, rich detail)
- **ChatGPT version** (~80–120 words, compact)
- **Light recipe** — key + fill + accent with positions, ratios, color temps
- **Color grade** — film stock + saturation + halation
- **Background** — specific environment
- **Preserve exactly** — identity preservation list
- **Avoid** — named AI failure modes (not generic)
- **Aspect ratio**
- **Subtle / Medium / Dramatic intensity variants**

### Marketing claim per pack

> *"25 prompts × 2 AI tools (ChatGPT + Nano Banana Pro) × 3 intensity variants = 150+ prompt blocks"*

### Quality bar

- Every prompt names a specific film stock + lens + shutter
- Every prompt references a specific photographer/director/aesthetic
- Every prompt includes negative prompts that name AI failure modes
- Every prompt has an evocative mood line
- Every prompt produces premium output in 4–6 generations
- **No generic "cinematic" without specificity**
