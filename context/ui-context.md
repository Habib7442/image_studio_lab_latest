# UI Context — Folio (imagestudiolab.com)

## Visual Language
Folio's design is **editorial-meets-product** (magazine-coded). It uses high-contrast typography, generous whitespace, sharp layouts, and realistic mockups. The interface feels premium and elite, avoiding the standard generic SaaS gradients or sci-fi circuit-board clichés.

## Color Tokens (Tailwind v4 theme variables)

| Token | Hex | Role |
| :--- | :--- | :--- |
| `background` | `#FAF7F2` | Primary background (Warm Cream) |
| `foreground` | `#15110D` | Primary text (Near-Black) |
| `accent` | `#FF4D2D` | Saturated Red-Orange (Primary CTAs, action highlights) |
| `sage` | `#4B5340` | Forest Sage (Secondary branding, borders, soft accents) |
| `muted` | `#6B5F52` | Soft text, subtitles, description details |
| `border` | `rgba(21, 17, 13, 0.08)` | Minimal dividers |

## Typography

| Style | Font Family | Usage |
| :--- | :--- | :--- |
| **Display** | `Fraunces` | Main headings, book quotes, brand wordmarks. Often italicized. |
| **Body** | `Inter` | User inputs, console text, labels, page descriptions. |
| **Code** | `JetBrains Mono` | Form fields, JSON code view, variables, product metadata. |

## Layout Patterns
- **Full Workspace Grid**: Splitting screen on desktop into a 1.2fr (Left Console) / 1.8fr (Right 3D Flipbook viewer) split layout.
- **Header Navigation**: High-visibility sticky header using background blur (`rgba(250, 247, 242, 0.92)`).
- **Dual vs. Single Spread**: 
  - **Desktop**: Renders double-page book layout.
  - **Mobile**: Automatically stacks pages into single-page viewport nodes for native swipe gestures.

## Key Component Patterns
1. **Interactive Hotspot**: A pulsing indicator overlays mapped visual assets. Hover/click slides up custom product details.
2. **Web Audio Paper Synthesis**: Generates sound based on page transition speeds, giving a physical weight to the digital lookbook.
3. **Palette Presets**: Brand-kit styles generated by AI:
   - **Earthy Sage**: Cream (`#FAF7F2`), Sage (`#4B5340`), Charcoal (`#252525`)
   - **Midnight Velvet**: Ink Black (`#0B0B0C`), Silver (`#E2E8F0`), Maroon (`#7B2D26`)
   - **Crimson Rose**: White (`#FFFFFF`), Rose (`#D946EF`), Gold (`#B8893E`)
